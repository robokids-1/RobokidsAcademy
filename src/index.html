üìö Robot Engineers Curriculum

1. Introduction to Robotics & Basic Electronics

Topics Covered:

  • What is a robot? Types and real-world applications
  • Robot parts: sensors, actuators, controllers
  • Safety in robotics
  • Basic electronics: circuits, LEDs, switches, batteries
Hands-On Project:

Build a Simple Flashlight: Students create a working flashlight using a battery, switch, and LED, learning about circuits and safety.

2. Makeblock mBot2 Building & Programming

Topics Covered:

  • Introduction to Makeblock mBot2 robot kit and components
  • mBot2 assembly: chassis, wheels, motors, and sensors
  • Understanding mBot2's sensors: color sensor, ultrasonic sensor, gyroscope
  • Mechanical assembly: frames, wheels, motors
  • Exploring basic sensors (light, ultrasonic, color detection)
  • Understanding how robots move and sense their environment
  • Introduction to mBlock 5 programming software
Hands-On Project:

Assemble and Program mBot2 Line-Following Robot: Students build a Makeblock mBot2 robot from scratch and program it to follow a black line on a track using the color sensor. Students will learn to calibrate sensors and adjust motor speeds for optimal performance.

3. VEX IQ Building

Topics Covered:

  • Introduction to VEX IQ platform and modular parts
  • Engineering design process: planning, building, testing
  • Adding attachments and customizing robots
  • Teamwork and collaboration in robotics
Hands-On Project:

Obstacle Course Challenge: Teams design and build a VEX IQ robot to navigate a simple obstacle course, adjusting attachments as needed.

4. VEX IQ Block Coding

Topics Covered:

  • Introduction to block-based programming (VEXcode)
  • Programming robot movements: forward, backward, turns
  • Using sensors in code (touch, distance)
  • Debugging and improving code
Hands-On Project:

Maze Navigation: Students program their VEX IQ robot to autonomously solve a maze using block coding and sensors.

5. Advanced Block Coding & Problem Solving

Topics Covered:

  • Advanced programming concepts: loops, conditionals, variables
  • Modular programming: breaking tasks into steps
  • Troubleshooting and debugging strategies
  • Creative problem-solving in robotics
Hands-On Project:

Sumo Bot Competition: Students code and modify robots to compete in a friendly sumo match, using logic and sensors to stay in the ring and push opponents out.

6. Introduction to Python Programming

Topics Covered:

  • Transition from block coding to text-based coding
  • Python basics: syntax, variables, loops, conditionals
  • Writing simple scripts to control robots
  • Comparing block code and Python for the same tasks
Hands-On Project:

Robot Dance Routine: Students write a Python script to make their robot perform a choreographed dance, using movement commands and timing.

7. Introduction to Raspberry Pi

Topics Covered:

  • What is a Raspberry Pi? Hardware overview and setup
  • Connecting peripherals: keyboard, mouse, display
  • Using GPIO pins to control LEDs and sensors
  • Running Python programs on Raspberry Pi
Hands-On Project:

Smart Light System: Students use Raspberry Pi and Python to build a light that turns on/off based on sensor input (e.g., light or motion sensor).

8. 3D Design & 3D Printing + Capstone Projects

Topics Covered:

  • Introduction to 3D printing technology and its role in robotics
  • Basics of 3D modeling using beginner-friendly software (e.g., Tinkercad)
  • Designing custom robot parts and accessories
  • Preparing and printing 3D models
  • Integrating 3D-printed parts into robotics projects
Hands-On Project:

Custom Robot Part Design: Students design and 3D print a unique part (e.g., a sensor mount or decorative shell) for their robot, then integrate it into their final capstone project.

Capstone Project & Showcase: Teams combine all skills learned to design, build, and present a creative robotics project, demonstrating coding, electronics, mechanical design, and 3D printing.

üìö Beginner Builders Curriculum

1. Introduction to Robots and Technology

Topics Covered:

  • What is a robot?
  • Robots in everyday life (vacuum robots, toy robots, Mars rovers)
  • Basic safety rules in the classroom
Hands-On Project:

Robot Show & Tell: Bring or draw a picture of a robot and explain what it does.

2. Exploring Simple Machines and Movements

Topics Covered:

  • Wheels and axles, levers, and gears
  • How simple machines help robots move
  • Building with blocks and connectors
Hands-On Project:

Make a Rolling Car: Build a simple car with wheels using LEGO, K'nex, or similar kits.

3. Building Basic Robots (No Coding)

Topics Covered:

  • Assembling robot parts (head, body, arms, wheels)
  • Understanding buttons and switches
  • Introduction to battery-powered toys and robots
Hands-On Project:

Create a Bristlebot: Make a tiny robot using a toothbrush head, motor, and battery that wiggles across the table.

4. Introduction to Sensors

Topics Covered:

  • What are sensors?
  • How robots "see" or "feel" (light, touch, sound)
  • Simple games to demonstrate sensing (e.g., clap to turn on the light)
Hands-On Project:

Sound-Activated Robot Toy: Use a toy or kit that responds to clapping or loud noises.

5. Introduction to Coding with Makeblock Codey Rocky

Topics Covered:

  • What is coding? Basic concepts using block programming
  • Exploring Codey Rocky's sensors and lights
  • Programming simple movements and sounds with Codey Rocky using Makeblock's mBlock app
Hands-On Project:

Codey Rocky Dance: Program Codey Rocky to perform a fun dance with lights and sounds using block coding.

6. Programmable Floor Robots (e.g., Bee-Bot)

Topics Covered:

  • Giving instructions to robots (forward, backward, turn)
  • Simple programming concepts (commands, order)
  • Problem-solving with robots
Hands-On Project:

Maze Challenge: Program a Bee-Bot or similar robot to navigate a simple floor maze.

7. Creative Robot Design

Topics Covered:

  • Imaginative thinking: What could robots do in the future?
  • Drawing and building creative robot models
  • Using craft materials to decorate robots
Hands-On Project:

Design Your Own Robot: Draw or build a robot out of recycled materials and explain its special job.

8. Mini-Showcase and Parent Day

Topics Covered:

  • Sharing what was learned
  • Presenting favorite projects
  • Celebrating creativity and teamwork
Hands-On Project:

Robot Parade: Display robots and drawings; each child presents their favorite project to the group and parents.

üìö Future Inventors Curriculum

1. Robotics Foundations & Engineering Concepts

Topics Covered:

  • Robotics in society and industry (career pathways, CTE connections)
  • Engineering design process
  • Overview of advanced robotics platforms (Arduino, Raspberry Pi, VEX EXP, VEX V5)
  • Safety, teamwork, and project management
Hands-On Project:

Robotics Research Presentation: Teams research a field of robotics (e.g., medical, industrial, AI) and present findings, connecting to CTE and real-world applications.

2. Mechanical Design & Prototyping

Topics Covered:

  • Advanced mechanical concepts: gear ratios, torque, chassis design
  • CAD basics for robotics (using Tinkercad, Fusion 360, or similar)
  • Prototyping and iterative design
Hands-On Project:

Custom Chassis Build: Design and build a robot chassis using VEX EXP or VEX V5 components; iterate based on performance tests.

3. Arduino Microcontroller Programming

Topics Covered:

  • Introduction to Arduino hardware and IDE
  • Wiring and using sensors/actuators (ultrasonic, IR, servo motors)
  • Writing and uploading Arduino C/C++ code
  • Data logging and debugging
Hands-On Project:

Obstacle-Avoiding Robot: Build and program an Arduino robot that uses ultrasonic sensors to detect and avoid obstacles.

4. Raspberry Pi & Physical Computing

Topics Covered:

  • Raspberry Pi setup (OS installation, terminal basics)
  • Python programming for robotics
  • Using GPIO pins for input/output and sensor integration
  • Networking and remote access
Hands-On Project:

Smart Home Prototype: Program a Raspberry Pi to automate lights or appliances based on sensor data (e.g., motion or temperature).

5. Advanced Programming & Control Systems

Topics Covered:

  • Structured programming: functions, classes, modularity
  • PID control and feedback systems
  • Introduction to ROS (Robot Operating System) concepts
  • Real-time data collection and analysis
Hands-On Project:

Line-Following Robot with PID: Implement a line-following algorithm with PID control using VEX V5 or Arduino.

6. VEX EXP & VEX V5 Robotics Competition Prep

Topics Covered:

  • VEX EXP and VEX V5 hardware and software ecosystem
  • Sensor integration and autonomous routines
  • Strategy and teamwork for competition
  • Documentation and engineering notebooks (CTE best practice)
Hands-On Project:

Competition Challenge: Build and program a robot to complete a VEX-style game or challenge, practicing both autonomous and driver control.

7. Artificial Intelligence & Robotics (AIR)

Topics Covered:

  • Introduction to AI: machine learning basics, computer vision
  • Using AI in robotics: object detection, decision making
  • Ethical considerations in AI and robotics
  • Real-world applications and case studies
Hands-On Project:

AI Object Recognition: Use a Raspberry Pi with a camera and Python (OpenCV or TensorFlow Lite) to build a simple object recognition system, integrating results with a robot's actions.

8. Capstone Project & Showcase

Topics Covered:

  • Project planning and proposal writing
  • Prototyping, testing, and iteration
  • Team presentations and technical communication
  • Reflecting on CTE and future career pathways
Hands-On Project:

Capstone Robotics Project: Students (individually or in teams) design, build, and program a robot that addresses a real-world problem or competition challenge, integrating mechanical, electronic, coding, and AI elements. Present final projects to peers, parents, and mentors.
